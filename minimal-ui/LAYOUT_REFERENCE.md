## Color Coding

### Status Colors
- âœ“ **ACCEPTED**: Green (`bg-emerald-500/20`, `text-emerald-400`)
- âš  **RUNNING**: Yellow (`bg-yellow-500/20`, `text-yellow-400`)
- âœ— **REJECTED**: Red (`bg-rose-500/20`, `text-rose-400`)

### Odds Colors
- **Home/Leg1**: Blue (`text-blue-400`)
- **Away/Leg2**: Orange (`text-orange-400`)

### Profit Colors
- **Positive**: Green (`text-emerald-400`)
- **Negative**: Red (`text-rose-400`)

### LED Status
- **CONNECTED**: Green with pulse
- **DISCONNECTED**: Red
- **ERROR**: Red with pulse
- **PROCESSING**: Yellow with pulse
- **STANDBY**: Gray

## Component Breakdown

### Left Column Components
1. **AccountPanel** (x2)
   - Balance display (large, green)
   - Sportsbook dropdown
   - URL input
   - Username/Password inputs
   - START TRADING button (green/red toggle)
   - Ping indicator
   - Status LED

2. **Configuration**
   - Tier stakes (3 inputs)
   - Profit range (min/max %)
   - Max minute (HT/FT)
   - Match filter (toggle: PREMATCH/LIVE/MIXED)
   - Market filter (6 checkboxes)

### Middle Column Components
1. **ExecutionHistory**
   - Table with 2-row format
   - 7 columns
   - Scrollable (max 350px)
   - Color-coded status badges
   - Blue/Orange odds distinction

2. **LiveScanner**
   - Table with 2-row format
   - 6 columns
   - Scrollable (max 350px)
   - Profit badge (green)
   - "Scanning..." indicator

### Right Column Components
1. **DailyPnL**
   - Large profit number
   - Start vs Current cards
   - Bar chart (recharts)
   - "From initial $XXX" text

2. **Logs**
   - Scrollable list (500-600px height)
   - Color-coded by level
   - Format: `[time] [LEVEL] message`
   - Dark background

## Responsive Behavior

### Desktop (lg+)
- 3 columns: 3-6-3 grid
- All panels visible side-by-side
- Tables scroll independently

### Tablet/Mobile
- Stacks vertically
- Full width panels
- Maintains all functionality
- Touch-friendly spacing

## Key Measurements
- Gap between panels: 3-4 units (0.75-1rem)
- Card padding: 3-4 units
- Table max height: 350px
- Logs height: 500-600px
- Max container width: 1920px
